## 📌 Project Overview

Pokedex Guide is a modern and responsive web application that allows users to explore Pokémon data in a clean and interactive interface.

The project fetches real-time data from the PokéAPI and displays it with dynamic UI elements, smooth animations, and a powerful live search system.

---

## 🚀 Features

✨ Real-time search (instant filtering)  
🎨 Dynamic card colors based on Pokémon type  
⚡ Fast API data fetching (async/await)  
📱 Fully responsive design  
🧩 Clean and scalable structure  
🖼️ High-quality Pokémon visuals  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox & Grid)  
- JavaScript (ES6+)  
- Fetch API  
- PokéAPI

## 🔍 How It Works

- Pokémon data is fetched from PokéAPI  
- Cards are dynamically created with JavaScript  
- Users can search by:
  - Name  
  - ID  
  - Type  
- Results update instantly without page reload
